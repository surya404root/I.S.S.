# ⚡ ISS - Internet Speed Scanner

Professional hacker-style internet speed testing toolkit.

## Features

- CLI Tool (Termux)
- PDF Reports 📄
- Graphs 📊
- Hacker UI 🟢
- Web Version 🌐

## Install
```
git clone https://github.com/surya404root/ISS-Speed-Scanner.git
cd ISS-Speed-Scanner
bash install.sh
```
## Run
```
python cli/cyber_speed.py
```
## Web

Open "web/index.html" or enable GitHub Pages.
