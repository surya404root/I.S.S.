# 🚀 I.S.S. - Internet Speed Scanner

A powerful and stylish internet speed testing tool made for Termux, Linux, and Web usage.  
This tool checks your internet performance with features like:

- 📥 Download Speed Test
- 📤 Upload Speed Test
- 📶 Ping Test
- 🌐 Network Information
- ⚡ Full Internet Check
- 📄 PDF Report Generation
- 🖥️ Web Version Support
- 🎨 Hacker-Style Interaction

---

## ✨ Features

- Simple and easy to use
- Works in Termux and Linux
- Stylish CLI interface
- Generates speed reports
- Beginner friendly
- Open-source

---

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/surya404root/I.S.S..git
cd I.S.S.
```
### Run the Tool 🔥 
```bash
bash iss.sh
```
